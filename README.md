# Toxicity_prediction_ml
Ml_based computational toxicity prediction system built using python and scikit_learn
# 🧪 Computational Toxicity Prediction System

## Project Overview
Machine learning project to predict chemical compound 
toxicity using Random Forest classification, inspired 
by real research work at CSIR-IITR.

## Dataset
- 500 chemical compounds
- 5 molecular features per compound
- Target: Toxicity classification (Toxic/Non-Toxic)

## Tech Stack
- Python
- Pandas & NumPy
- Scikit-learn (Random Forest)
- Matplotlib & Seaborn

## Project Pipeline
1. Data Generation & Loading
2. Exploratory Data Analysis
3. Data Cleaning & Preprocessing
4. Model Training & Evaluation
5. Feature Importance Analysis
6. Correlation Heatmap

## Key Outputs
- Toxicity distribution chart
- Feature importance ranking
- Correlation heatmap
- Model evaluation report

## Skills Demonstrated
- End-to-end ML pipeline
- Data preprocessing
- Classification modeling
- Data visualization
- Research domain knowledge
